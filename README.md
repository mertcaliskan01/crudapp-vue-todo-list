# Todo App

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```


### INFO

– store for Vuex (src/store)
– router for Vue Router (src/router.js)
– bootstrap with CSS
– vue-fontawesome for icons
